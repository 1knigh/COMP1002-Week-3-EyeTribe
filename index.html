The New and Improved

The Eye Tribe software allows users to interact with their systems in a completely different way. The software allows the user to use not their hands, but their eyes as a controller. This invention can support those who have trouble gaming with their hands. The technology achieved through The Eye Tribe can be used not only to innovate gaming but to innovate the world. The invention of eye trackers will shine a new light on technology.

How it works

The Eye Tribe uses a device called the Eye Tracker or gaze tracker. This device calculates the gaze point of a user as they look around. The device can be combined with other input devices like the mouse, keyboard, touch etc. The users eyes are tracked by the sensor which is place below the screen of the device. The eye gaze coordinates are calculated with respect to a screen the user is looking at. The system works for a variety of different users thanks to the calibration system. Before the user games with the device they must complete a colibration process. Everyone has different eyes and the eye tracker device can store many different users eye characteristics.

Make Way For The Future

The Eye Tribe software does stop at satisfying the user. It collects data when the user is not gaming or using the device. The product uses the data and it is sent to the development team to fix or improve the device. The Eye Tribe's software is not made private for the corporations benefits. The code for the program is made public for all those who are fascinated by the creation of a different breed of gaming.
